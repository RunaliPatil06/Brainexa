# 🧠 Brainexa

> An AI-powered conversational assistant built with the MERN stack, inspired by ChatGPT.

Brainexa is a modern AI chatbot that enables users to have natural conversations with an intelligent assistant. It features a clean, responsive interface, real-time AI responses, and conversation management, providing an engaging chat experience.

---

## 🚀 Features

- 💬 Real-time AI conversations
- 🧠 Powered by Groq API (Llama Models)
- 📜 Chat history management
- ➕ Create new conversations
- 🗑️ Delete chats
- 📱 Fully responsive design
- 🌙 Modern and minimal UI
- ⚡ Fast AI responses
- 🔒 Secure backend with REST APIs
-  User Authentication
---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Brainexa.git
```

### Navigate to the project

```bash
cd Brainexa
```

### Install Frontend Dependencies

```bash
cd Frontend
npm install
```

### Install Backend Dependencies

```bash
cd ../Backend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **Backend** directory.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

GROQ_API_KEY=your_groq_api_key
```

---

## ▶️ Running the Application

### Start Backend

```bash
cd Backend
npm run dev
```

### Start Frontend

```bash
cd Frontend
npm run dev
```

The application will be available at:

```
Frontend : http://localhost:5173

Backend : http://localhost:5000
```
